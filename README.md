Проект приложение банка, полностью написан full vanilla js. БД, работа с rest api (GET, PUT, POST, DELETE). Так же есть авторизация пользователя(с использованием токена), работа с cookie. Возможность перевести другому человеку деньги, показ всех твоих транзакций. Анимированный график транзакций. Собран проект на webpack
